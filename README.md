# CLI JSON-to-CSV Converter

Provide a command-line interface (CLI) utility that transforms structured JSON data files into valid CSV files. This utility allows data operators to easily format API payloads and JSON documents for ingestion into spreadsheet tools and legacy databases that require delimited text structures.

## Features
- Standard CLI interface to transform flat JSON arrays of uniform or heterogeneous objects into RFC 4180-compliant CSV files.
- Customizable field delimiter selection via CLI options, supporting common delimiters and tab characters.
- Automatic hierarchical structure flattening using dot-notation column headers and nested array serialization.

## Getting Started

### Prerequisites
Python 3.12 runtime and standard package manager.

### Installation & Build
```sh
pip install --target=/workspace/.sdlc_deps -r requirements.txt 2>/dev/null || true
python -m compileall -q -x '(^|/)\.sdlc_deps(/|$)' .
```

## Usage
```sh
python -m src.cli --input input.json --output output.csv --delimiter ","
```
